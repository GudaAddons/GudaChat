# GudaChat

A clean, minimalistic chat replacement for World of Warcraft. GudaChat strips away the clutter of Blizzard's default chat frame and replaces it with a modern, streamlined interface, while keeping all the functionality you need.

![GudaChat overview](https://media.forgecdn.net/attachments/description/null/description_3e6483bb-796a-48c6-acab-454e9195a170.png)

## Features

### Hover-Reveal Header Bar

A compact header bar appears on hover with quick-access icons for tab switching, combat log, whispers, chat type selection, channels, history, and settings. Stays out of your way until you need it.

![Header bar with icons visible on hover](https://media.forgecdn.net/attachments/description/null/description_0d63d043-d3e8-4672-bc64-bffdc6c4ff61.png)

### Emoji Support

Over 30 built-in emojis that render inline in chat messages. Type shortcuts like `:)` `:fire:` `:skull:` or use the emoji picker button in the input bar. Adjustable emoji size.

![Chat with emojis and emoji picker open](https://media.forgecdn.net/attachments/description/null/description_460fc6da-04ba-4715-a7f0-e22d889cdc4c.png)

### Chat History

Messages persist across sessions and reloads. Browse, search, and filter history by channel. Copy entire conversations to clipboard with one click.

![History viewer with search and channel tabs](https://media.forgecdn.net/attachments/description/null/description_05957f81-4b66-430d-9b7b-a865ee3046d5.png)

### Copyable Links

URLs in chat are automatically highlighted and clickable. Click any link to open a copy popup — no more manually retyping URLs.

### Class-Colored Names & Player Levels

Player names are colored by class. Optional level indicators show difficulty-colored level brackets next to each player's name.

![Chat showing class colors and level brackets](https://media.forgecdn.net/attachments/description/null/description_94b7d90d-79a8-4045-a290-6995a133c94f.png)

### Dedicated Whisper Tab

Enable a separate whisper tab to keep private messages organized. Incoming whispers trigger a blinking notification on the header icon.

### Combat Log Filtering

Quick-filter the combat log with subtabs: **All**, **My Actions**, or **What Happened to Me?**

### Customizable Settings

Access settings with `/gc` or the header gear icon. Three tabs of options:

- **General** — Lock position, disable fading, hide scrollbar, timestamps, input bar position, whisper tab toggle
- **Messages** — Class colors, player levels, copyable links, emoji toggle and size
- **History** — Enable/disable, max message limit, clear history

![Settings panel](https://media.forgecdn.net/attachments/description/null/description_ba45690f-1b5f-4498-8732-41ce7f8ec80b.png)

### Additional Details

- Custom context menu (right-click the tab name) for renaming, creating, and removing chat windows, font size, and background color
- Sleek custom scrollbar with scroll-to-bottom button
- Drag-to-move via the header bar with lockable position
- Works across Classic, Cataclysm, Wrath, and Retail

## Slash Commands

- `/gc` or `/gudachat` — Open settings
